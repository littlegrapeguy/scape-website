The new tab page with simplicity in mind.

Scape is a simple, customisable and open source new tab page.






Feature rich, but simple.

We believe that a new tab page should have many customisation features while still keeping it simple. That exact vision is what Scape is today.

Free and open source

Scape is built on open source code, meaning that code is open to anyone. It also completely free.

Make Scape yours

Tweak Scape's wide range of settings to get Scape to look how you like it.

Accurate Weather

Scape has a fast and accurate weather service to make sure you know what's waiting for you outside.

Cross Platform

Scape is available to download on the Chrome Webstore, Mozilla Add-ons, and Edge Extensions.
